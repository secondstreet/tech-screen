Front-End CSS Exercise

Using only styling, please implement the "author" component in a new CodePen. The mockups are at the bottom of this page, below the code block.

Guidelines & Rules

  • Do not change the HTML, and do not write any JavaScript.
  • The component should be responsive.
  • Feel free to configure the CodePen to use Sass, PostCSS, etc.
  • Feel free to look up references like MDN, CSS Tricks, Stack Overflow, and others.

Instructions

  1. Create a CodePen
  2. Paste the code below into the HTML area of the CodePen
  3. Implement the "author" component in the CSS area of the CodePen
